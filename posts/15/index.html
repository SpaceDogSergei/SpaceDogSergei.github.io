モデル設定

テンパイ時のモデルと同じルーティンで考える。パラメータも同じく：

  • \(t\): 巡目。1 から 21 までの整数
  • \(p_0\): 1牌あたりの自分のツモ率
  • \(\widetilde{p}_0\): 1牌あたりの自分の出アガリ率
  • \(p_1\): 1牌あたりの下家のツモ率
  • \(\widetilde{p}_1\): 1牌あたりの下家の出アガリ率
  • \(p_2\): 1牌あたりの対面のツモ率
  • \(\widetilde{p}_2\): 1牌あたりの対面の出アガリ率
  • \(r_1\): 現在自分が切ろうとする牌の下家への放銃率
  • \(r_2\): 現在自分が切ろうとする牌の対面への放銃率
  • \(w_{0,t}\): \(t\) 巡目で自分のツモ率
  • \(\widetilde{w}_{0,t}\): \(t\) 巡目で自分の出アガリ率
  • \(w_{1,t}\): \(t\)巡目で下家のツモ率
  • \(\widetilde{w}_{1,t}\): \(t\) 巡目で下家の自分からの出アガリ率
  • \(\widehat{w}_{1,t}\): \(t\) 巡目で下家の対面からの出アガリ率
  • \(w_{2,t}\): \(t\)巡目で対面のツモ率
  • \(\widetilde{w}_{2,t}\): \(t\) 巡目で対面の自分からの出アガリ率
  • \(\widehat{w}_{2,t}\): \(t\) 巡目で対面の下家からの出アガリ率
  • \(A_t\): \(t\) 巡目で攻める時の期待収支
  • \(B_t\): \(t\) 巡目でオリる時の期待収支
  • \(W_0\): 自分のツモ時の期待収支
  • \(\widetilde{W}_0\): 自分の出アガリ時の期待収支
  • \(W_1\): 下家のツモ時の期待収支（マイナス）
  • \(\widetilde{W}_1\): 下家の自分から出アガリ時の期待収支（マイナス）
  • \(\widehat{W}_1\): 下家の自分以外から出アガリ時の期待収支（マイナス）
  • \(W_2\): 対面のツモ時の期待収支（マイナス）
  • \(\widetilde{W}_2\): 対面の自分から出アガリ時の期待収支（マイナス）
  • \(\widehat{W}_2\): 対面の自分以外から出アガリ時の期待収支（マイナス）

新たに

  • \(s\): 自分が次の巡目でテンパイできる確率
  • \(C_t\): \(t\) 巡目で攻める時の期待収支（一向聴時）

を追加する。

自分以外に一人テンパイの場合：

\begin{eqnarray} C_t &=& \begin{cases} r_1 \widetilde{W}_1 + (1 - r_1) \widetilde{p}_1 W_1 + (1 - r_1) (1 - \widetilde{p}_1) s A_t + (1 - r_1) (1 - \widetilde{p}_1) (1 - s) C_{t + 1} & (1 \leq t \lt 21) \\
0 & (t = 21) \end{cases} \end{eqnarray}

ただし \(w’_{0, t+1}\) と \(\widetilde{w}’_{0, t+1}\) はテンパイ時のパラメータである。この漸化式を解いて

$$C_t = (1 - \phi^{21 - t})\frac{r_1 \widetilde{W}_1 + (1 - r_1) \widetilde{p}_1 W_1 + (1 - r_1) (1 - \widetilde{p}_1) s A_t}{1 - \phi}$$

が得られる。ただし、 \(\phi = (1 - r_1) (1 - \widetilde{p}_1) (1 - s)\) である。
