押し引きや山読みなどについてはまだデータが足りないので、しばらく個別の手役の研究が続く。

最初に五門斉を選んだ理由は、出現率の高さはもちろん、研究対象としては三色系よりも結論出しやすそうである。端牌と字牌のアガリやすさは正確なデータがないのでとりあえず無視する。また、特に説明がなければ場況は平場とする。

テンパイ時の選択

出アガリ可能なカンチャン　と　ツモ専のリャンメン

　　

以前のモデルをそのまま使ってみる。パラメータは以下のように近似する。

  • \(p_1 = \widetilde{p}_1 \fallingdotseq \frac{1}{34}\)
  • \(p_2 = \widetilde{p}_2 \fallingdotseq \frac{1}{34}\)
  • \(r_1 = r_2 \fallingdotseq \frac{1}{34}\)
  • \(W_1 = -14.94, \widetilde{W}_1 = -13.69, \widehat{W}_1 = -8\)
  • \(W_2 = -14.94, \widetilde{W}_2 = -13.69, \widehat{W}_2 = -8\)

相手のアガリ点は、例のサイトから全プレイヤーの平均アガリ素点が \(13.69\) で、ツモ率と不求人の出現頻度の比は \(3.02\) から計算した。また、自分のアガリに関するパラメータは

  • \(p_0 = \widetilde{p}_0 \fallingdotseq \frac{1}{34}, W_0 = 51, \widetilde{W}_0 = 32\) （出アガリ可能なカンチャン）
  • \(p_0 \fallingdotseq \frac{2}{34}, \widetilde{p}_0 \fallingdotseq 0, W_0 = 48, \widetilde{W}_0 = 0\) （ツモ専のリャンメン）

断然出アガリ可能なカンチャンのほうが有利である。

出アガリ可能なカンチャン　と　片方ツモ専のリャンメン

　　

これは数式を使う必要もなく、明らかに片方ツモ専のリャンメンのほうが全てにおいて有利である。よってを切るべきである。

出アガリ可能な高打点のカンチャン　と　片方ツモ専のリャンメン

　　

自分のアガリに関するパラメータは

  • \(p_0 = \widetilde{p}_0 \fallingdotseq \frac{1}{34}, W_0 = 69, \widetilde{W}_0 = 38\) （出アガリ可能なカンチャン）
  • \(p_0 \fallingdotseq \frac{2}{34}, \widetilde{p}_0 \fallingdotseq \frac{1}{34}, W_0 = 49, \widetilde{W}_0 = 32\) （ツモ専のリャンメン）

リャンメンのほうがちょっと有利である。仮にが一枚多く切られていると

とわずかながら逆転する。よってとの残量が同じならを切るが、のほうが一枚ぐらい少ないならを切るのが有利である。

ツモり三暗刻（１）

　　

自分のアガリに関するパラメータは

  • \(p_0 = \widetilde{p}_0 \fallingdotseq \frac{1}{34}, W_0 = 114, \widetilde{W}_0 = 39\) （シャンポン）
  • \(p_0 \fallingdotseq \frac{2}{34}, \widetilde{p}_0 \fallingdotseq \frac{2}{34}, W_0 = 54, \widetilde{W}_0 = 33\) （リャンメン）

これもリャンメンのほうがちょっと有利であるが、同じく仮にが一枚少ないとすると

と逆転する。よって、、との残量が同じならを切るが、かのほうが一枚ぐらい少ない、もしくは字牌が期待できる場況ならを切るのが有利である。

ツモり三暗刻（２）

　　

自分のアガリに関するパラメータは

  • \(p_0 = \widetilde{p}_0 \fallingdotseq \frac{1}{34}, W_0 = 126, \widetilde{W}_0 = 43\) （シャンポン）
  • \(p_0 \fallingdotseq \frac{2}{34}, \widetilde{p}_0 \fallingdotseq \frac{2}{34}, W_0 = 54, \widetilde{W}_0 = 33\) （リャンメン）

すでに微妙な感じ。もし三色三節高ではなく三同刻だったら

断然出アガリ可能なカンチャンのほうが有利である。

まとめ

期待収支が高い順で並ぶと：

  • 出アガリ可能な超高打点の愚形
  • 片方ツモ専の好形
  • 出アガリ可能な高打点の愚形
  • 出アガリ可能な愚形
  • ツモ専の好形

やっぱりツモが大きいという感じの結果であった。
